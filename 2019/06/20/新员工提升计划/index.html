新员工提升计划

发表于 2019-06-20 | 更新于 2019-07-18

新员工提升计划:
一.

1. 了解公司和部门制度文化
2. 熟悉本岗位职责,工作内容,工作流程
3. 完成新员工入职在线测试

二. (2019.6.21)用原生html5+css3+javascript重构一个静态页面

1. 要求移动端优先,采用响应式布局进行页面搭建,(页面结构,命名规范,标签等使用合理)
2. phpstudy(php+mysql+apache)(本地配置域名站点)
3. vscode工具使用(插件)
常用插件: HTML Snippets ,Auto Close Tag,Auto Rename Tag,chinese,easy sass
工具地址:https://code.visualstudio.com/
1. ps工具使用(截图,颜色,字体等)
2. 浏览器(谷歌,火狐.....)调试代码

三. (2019.7.4)熟悉常用框架(Pintuer/BootStrap)(5天)

1. 合理使用栅格布局,熟悉文档
2. 代码规范sass
3. gitee代码管理工具的正确使用(分栏目提交)
4. cmder
5. 重构一套优化模板

四. (2019.7.6-7.20)dedecms添加后台(5天)

1. 下载学习使用dedecms,熟练使用后台添加栏目,分类等
2. 将静态代码添加后台遍历内容
3. 学习使用调用商务通及留言板等
4. MySQL熟练掌握,Navicat工具的使用
5. 学习FTP工具使用

五. (2019.7.22-8.1)使用(Pintuer/BootStrap)框架重构一套优化模板

1. 要求提高代码质量及重构速度
2. 使用gitee托管代码,
3. 布局合理等

六. (2019.8)

1. 熟悉lmzg后台文件及栏目结构
2. 逐步学习操作后台文件的修改,上传,栏目更新等
3. 记录lmzg关键词排名情况及网站数据分析统计
4. lmzg网站友链交换
5. 熟悉thinkphp6.0文档
6. 熟悉手机端界面搭建,熟悉mintui框架,做一套手机站
