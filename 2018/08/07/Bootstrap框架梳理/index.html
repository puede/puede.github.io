Bootstrap简介:

Bootstrap 是一个用于快速开发 Web 应用程序和网站的前端框架。Bootstrap 是基于 HTML、CSS、JAVASCRIPT
的。

优点:
移动设备优先 浏览器支持 易上手 响应式布局

1. 基本的html结构

<meta>标签必须放在最前面
META标签用来描述一个HTML网页文档的属性，例如作者、日期和时间、网页描述、关键词、页面刷新等
